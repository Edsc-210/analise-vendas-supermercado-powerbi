\# Análise de Vendas – Supermercado



Este projeto tem como objetivo analisar dados de vendas de um supermercado utilizando \*\*SQL\*\* e \*\*Power BI\*\*, com foco em geração de insights para tomada de decisão.



\##  Ferramentas utilizadas

\- MySQL

\- Power BI

\- SQL



\##  Principais análises realizadas

\- Faturamento total do período

\- Faturamento por produto

\- Faturamento por categoria

\- Volume de vendas por produto

\- Análise de volume x faturamento



\##  Principais insights

\- O faturamento total do período analisado foi de R$ 209,80.

\- O produto com maior volume de vendas foi a Cerveja Lata.

\- O produto com maior faturamento foi o Arroz 5kg.

\- A categoria Alimentos concentra a maior parte da receita.

\- Existem produtos com alto volume de vendas, mas baixo faturamento unitário.



\##  Estrutura do projeto

\- `sql/` → scripts de criação, inserção e consultas

\- `powerbi/` → dashboard final

\- `imagens/` → visual do dashboard



\##  Observações

Os dados utilizados são simulados e têm finalidade exclusivamente educacional.
